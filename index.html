Pengertian Javascript


Javascript adalah bahasa Pemrograman berbasis text yang sering di pake dalam pembuatan website. Mengapa Javascript sering di gunakan dalam pembuatan website karena:

1.Interkatif
2.Responsif
3.Banyak Fiturnya

Dengan keunggulan keunggulan Javascript tersebut maka web developer atau programer lebih memilih memakai Javascript untuk membuat website mereka.
